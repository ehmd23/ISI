Introduction

Dans ce TP, l’objectif est de déployer un cluster Hadoop distribué de 4 nœuds composé d’un NameNode et de 3 DataNodes à l’aide de VirtualBox. Le cluster sera configuré avec HDFS pour le stockage distribué, YARN pour la gestion des ressources, puis validé par l’exécution d’un job MapReduce.

Objectif du TP : installer un cluster Hadoop complet (1 master + 3 workers), configurer les communications réseau, activer HDFS et YARN, puis exécuter un traitement MapReduce permettant de calculer le chiffre d’affaires total par ville.

Ce TP s’adresse à un niveau novice et met l’accent sur une démarche pas à pas : préparation des machines virtuelles, configuration système, installation logicielle, configuration du cluster puis validation fonctionnelle.

1.2 Machines virtuelles à créer

Créez les 4 machines virtuelles suivantes dans VirtualBox :

Rôle Nom de la VM RAM CPU Disque Carte 1 Carte 2
Master hadoop-master 2 Go 2 30 Go NAT Host-only
DataNode 1 hadoop-datanode1 2 Go 2 30 Go NAT Host-only
DataNode 2 hadoop-datanode2 2 Go 2 30 Go NAT Host-only
DataNode 3 hadoop-datanode3 2 Go 2 30 Go NAT Host-only

1.3 Réseau VirtualBox

Créez un adaptateur Host-only depuis Fichier → Outils → Gestionnaire de réseaux hôte, puis créez vboxnet0 et désactivez le serveur DHCP.

Important : installez Ubuntu 24.04 LTS sur les 4 machines avant de commencer la configuration Hadoop.

2. Configuration réseau et hostnames

Les opérations de cette section doivent être exécutées sur les 4 machines virtuelles.

2.1 Configuration des adresses IP

sudo nano /etc/netplan/01-netcfg.yaml

Collez la configuration suivante puis adaptez l’adresse IP selon la machine :

network:
  version: 2
  ethernets:
    enp0s3:
      dhcp4: true
    enp0s8:
      dhcp4: false
      addresses:
        - 192.168.56.10/24     # master → 10
        # - 192.168.56.11/24   # datanode1 → 11
        # - 192.168.56.12/24   # datanode2 → 12
        # - 192.168.56.13/24   # datanode3 → 13
sudo netplan apply

2.2 Définition des hostnames

sudo hostnamectl set-hostname master        # Sur master
sudo hostnamectl set-hostname datanode1     # Sur datanode1
sudo hostnamectl set-hostname datanode2     # Sur datanode2
sudo hostnamectl set-hostname datanode3     # Sur datanode3

2.3 Mise à jour du fichier /etc/hosts

sudo nano /etc/hosts

Ajoutez les entrées suivantes sur chacune des 4 machines :

192.168.56.10   master

3. Création de l’utilisateur Hadoop

À exécuter sur les 4 machines :

sudo adduser --disabled-password --gecos "" hadoop

4. Installation de Java et Hadoop

Installez Java 21 puis téléchargez Hadoop sur les 4 machines :

sudo apt update && sudo apt install openjdk-21-jdk -y

cd ~
wget https://archive.apache.org/dist/hadoop/common/hadoop-3.4.3/hadoop-3.4.3.tar.gz
tar -xzf hadoop-3.4.3.tar.gz
sudo mv hadoop-3.4.3 /opt/hadoop
sudo chown -R hadoop:hadoop /opt/hadoop

5. Variables d’environnement

Éditez le fichier ~/.bashrc sur les 4 machines :

nano ~/.bashrc

Ajoutez à la fin :

export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export HADOOP_HOME=/opt/hadoop
export HADOOP_CONF_DIR=$HADOOP_HOME/etc/hadoop
export PATH=$PATH:$HADOOP_HOME/bin:$HADOOP_HOME/sbin
export HADOOP_OPTS="-Djava.library.path=$HADOOP_HOME/lib/native"

Puis appliquez la configuration :

source ~/.bashrc

6. Configuration SSH sans mot de passe

Cette étape est réalisée uniquement sur le master :

ssh-keygen -t rsa -P "" -f ~/.ssh/id_rsa

ssh-copy-id hadoop@datanode1
ssh-copy-id hadoop@datanode2
ssh-copy-id hadoop@datanode3

Test de connexion :

ssh datanode1
Si la connexion SSH vers les DataNodes fonctionne sans demande répétée de mot de passe, la communication inter-nœuds est correctement configurée.

11.4 Code source Java : TotalSalesByCity.java

mkdir -p src/main/java/sn/ehmd
nano src/main/java/sn/ehmd/TotalSalesByCity.java
package sn.ehmd;

import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.DoubleWritable;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

import java.io.IOException;

public class TotalSalesByCity {

    public static class SalesMapper extends Mapper<LongWritable, Text, Text, DoubleWritable> {
        private final Text city = new Text();
        private final DoubleWritable amount = new DoubleWritable();

        public void map(LongWritable key, Text value, Context context) throws IOException, InterruptedException {
            String line = value.toString().trim();
            if (line.isEmpty()) return;

            String[] fields = line.split("\\t");
            if (fields.length >= 5) {
                try {
                    String cityName = fields[2].trim();
                    double saleAmount = Double.parseDouble(fields[4].trim());

                    city.set(cityName);
                    amount.set(saleAmount);
                    context.write(city, amount);
                } catch (Exception e) {
                    System.err.println("Erreur parsing : " + line);
                }
            }
        }
    }

    public static class SalesReducer extends Reducer<Text, DoubleWritable, Text, DoubleWritable> {
        private final DoubleWritable total = new DoubleWritable();

        public void reduce(Text key, Iterable<DoubleWritable> values, Context context)
                throws IOException, InterruptedException {
            double sum = 0.0;
            for (DoubleWritable val : values) {
                sum += val.get();
            }
            total.set(sum);
            context.write(key, total);
        }
    }

    public static void main(String[] args) throws Exception {
        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "Total CA par ville");

        job.setJarByClass(TotalSalesByCity.class);
        job.setMapperClass(SalesMapper.class);
        job.setReducerClass(SalesReducer.class);

        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(DoubleWritable.class);

        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}

12. Résultat attendu

Le programme MapReduce doit agréger les montants de vente par ville à partir du fichier purchases.txt. Le fichier de sortie produit dans HDFS doit contenir une ligne par ville avec le montant total correspondant.
