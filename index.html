TP1 Hadoop 3.4.3 Pseudo-distribué

Installation et configuration d'Apache Hadoop en mode pseudo-distribué

Ce premier TP présente l'installation de Hadoop sur une seule machine en mode pseudo-distribué, avec la configuration de Java, SSH, HDFS, YARN et l'exécution d'un premier job de test.

  • Installation de Java OpenJDK 21
  • Configuration SSH et utilisateur hadoop
  • Configuration de HDFS, MapReduce et YARN
  • Validation avec un job MapReduce d'exemple
Ouvrir le TP1
TP2 Cluster 4 nœuds VirtualBox

Déploiement d'un cluster Hadoop 4 nœuds sur VirtualBox

Ce deuxième TP étend le précédent vers une architecture réellement distribuée, avec un master et trois workers, la configuration réseau inter-VM et l'exécution d'un job MapReduce métier de calcul du chiffre d'affaires par ville.

  • Création des 4 machines virtuelles
  • Configuration réseau, hostname et SSH
  • Déploiement de HDFS et YARN sur plusieurs nœuds
  • Exécution du programme Java TotalSalesByCity
Ouvrir le TP2
TP3 AWS EC2 Cloud

Déploiement d'un cluster Hadoop 4 nœuds sur AWS EC2

Ce troisième TP transpose le cluster Hadoop distribué vers le cloud AWS, avec la création des instances EC2, la configuration du Security Group, la mise en réseau des nœuds et l'exécution du même job MapReduce dans un environnement scalable.

  • Lancement et sécurisation de 4 instances EC2
  • Configuration réseau privée, hosts et SSH
  • Déploiement de HDFS et YARN dans AWS
  • Exécution du job MapReduce de calcul du CA par ville
